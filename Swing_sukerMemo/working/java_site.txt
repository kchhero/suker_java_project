Front End 응용프로그램을 기존의 웹 기반 언어를 사용하지 않고 Java 프로그래밍 언어로 신속하게 
개발하고 관리할 수 있게 하여 전체적인 개발 노력을 단축시키고 엔터프라이즈 급 어플리케이션을 
신속하게 개발할 수 있게 한다.

함으로써, 수강생들의 Career Path 계발에 미래 지향적인 시사점을 부여할 수 있다.
